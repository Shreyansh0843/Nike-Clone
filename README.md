### Run Locally

To run **responsive-nike-clone** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Shreyansh0843/Nike-Clone.git
```

Windows:

```bash
git clone https://github.com/Shreyansh0843/Nike-Clone.git
```
